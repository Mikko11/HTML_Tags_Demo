Before we proceed to main topic let us know first what HTML TAGS is. So when we say HTML TAGS it is the building blocks of an HTML document. They are used to structure and define the content and elements within a webpage. HTML TAGS are enclose in angle brackets (<and>) and consist of an opening tag and a closing tag, although some tags are self-closing and dont't require a closing tag.



There are several HTML TAGS that can be used for formatting and styling text.

Here are some commonly used HTML TAGS for text:
  1. < h1 > to < h6 > tag: Headings of different levels,where <h1> represents the highest level and <h6> represents the lowest level.

  6. < strong > tag: Indicates strong emphasis, typically rendered in bold. Just like this

  7. < em > tag: Indicates emphasis, typically rendered in italic.

  15. < pre > tag:
